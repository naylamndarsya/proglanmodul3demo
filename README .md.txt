1. Dalam program ini masukkan opsi 1 yaitu add task untuk memasukkan tugas didalam daftar list
2. Pada tahap ini user dapat memilih untuk melihat list tugas atau ingin menghapus list tugas
3. Bila memilih hapus (remove) maka list tugas akan terhapus dan dapat dilihat saat memeriksa di opsi ke3 yaitu list task
4. Setelah selesai memilih opsi apa yang ingin dilakukan maka user dapat keluar atau mengakhiri proses dengan memilih opsi 4 yaitu exit
